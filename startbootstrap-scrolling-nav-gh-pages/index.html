Certified vs. Denied Predictive Models

To demonstrate machine learning on the green card data from 2015-2018, the team cleaned data pertaining to over 70 different countries (100 or more starting rows of data) and 10 different world regions. Models were generated from a variety of different factors in order to predict whether a green card application would be CERTIFIED or DENIED. The following factors where used in the machine learning predictive models:

  • Employer Number of Employees
  • Yearly Wage Offer
  • Number of Days Til Decision
  • World Region
  • Applicant Education Level
  • Class of Admission (ex. H-1B)
  • Month of Application Case Received
  • Employer State Region

Future work in this area would be to incorporate a user input system that would allow people to gauge their application status by inputing their characteristics into the machine learning model.
